## Infrastructure

Toutes ces applications sont hébergées sur un serveur personnel et exposées via un tunnel Cloudflare pour un accès sécurisé en HTTPS.

### Technologies utilisées

- **Docker** : Conteneurisation des applications
- **Cloudflare Tunnel** : Exposition sécurisée des services
- **Nginx** : Reverse proxy
- **PostgreSQL** : Base de données
- **Streamlit** : Interface web (Chercher Voiture)
